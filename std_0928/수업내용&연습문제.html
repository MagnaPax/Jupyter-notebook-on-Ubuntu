카이제곱 검정의 해석¶

공식) 카이 제곱 검증 과정에서 카이 제곱을 계산하는 법, 공식

(관찰빈도 0) 에서 기대빈도 E를 뺀 후 그것을 제곱해서 (기대빈도 E) 로 나눈 값을 모두 합친 값이다. => Pvalue 를 계산

카이제곱검증 예문¶

  • 적합도 검정과 독립 검정으로 나뉜다.

  • 적합도 검정(Goodness of fit test) : 관측된 데이터가 예측한 분포를 따르는지 검정하는 방법이다.

  • 독립성 검정(Test of independence): 관측값들이 다수의 인자들에 의해 분할 되어 있는 경우 그 인자들의 관찰 값에 영향을 주고 있는지 아닌지를 검정하는 방법.

P-value < 0.05 = 대립가설¶

  • 내가 주장하고 싶은 것은 대립가설인데 귀무가설은 대립가설에 반대되는 가설이다.
  • p-value를 기준으로 0.05 보다 작으면95% 신뢰수준하에서 유의한 관계가 있다 그래서 대립가설 채택
  • 즉 내가 주장하고 싶은 가설을 채택한다.

#---> 독립표본 T 검정
#귀무가설 : 학생들의 평균키는 175cm이다.
#대립가설 : 학생들의 평균키는 175cm가 아니다.
import numpy as np
from scipy import stats
np.random.seed(1) #랜덤값 고정하면 항상 같은 값이 나온다.
heights = [180 + np.random.normal(0,5) for a in range(20)]
#print(heights)
print("------------------------------------------------------------------")
# 독립표본 T 검정에 대한 라이브러리 Scipy 통계라이브러리에서 제공 => ttest_1samp
# 독립표본 T 검정
result = stats.ttest_1samp(heights, 175)
print(result)
print("단일표본 T-검정  검정통계량: %.3f, pvalue: %.3f" % result)
#-----------------------------------------------------
# pvalue: 0.003 < 0.05 보다 작기 때문에 통계적으로 유의하고 내가 채택하고 싶은 대립가설이 채택
statistic,pvalue = result
if pvalue < 0.05:
    print("대립가설 채택")
else:
    print("귀무가설 채택")

# 귀무가설 : 학생들의 평균키는 같다.(상관관계가 없다.)

from scipy import stats
np.random.seed(1) #랜덤 시드 고정

# np.random.normal(평균,표준편차)

group1=[170+np.random.normal(0,5) for a in range(20)] 
# 그룹2 : 평균 175, 표준편차 10
group2=[175+np.random.normal(0,10) for a in range(20)] 

# numpy를 사용해서 각 그룹의 평균을 구한다.
print(group1)
print(group2)
print("*****************************************************")

print("group1의 평균 :",np.mean(group1))
print("group2의 평균 :",np.mean(group2))

import numpy as np
from scipy import stats
np.random.seed(1)
# 복용전 : 평균 60, 표준편차 5
# 여기서 60을 중심으로 체중 데이터 20개가 만들어 진다.
before = [60 + np.random.normal(0,5) for a in range(20)]
print(before)
# 복용후 : 평균 : 복용전체중 x 0.99 , 표준편차 0.02
after = [w * np.random.normal(0.99, 0.02) for w in before]
print(after)
print("---------------------------------------------------------")
# 대응 표본 t검정 - ttest_rel란 함수에다가 복용전과 복용후의 값을 인자로 입력 
result = stats.ttest_rel(before, after)
print(result)

print("단일표본 T-검정  검정통계량: %.3f, pvalue: %.3f" % result)
statistic,pvalue = result
if pvalue < 0.05: 
    print("대립가설 채택")
else:
    print("귀무가설 채택")

#ANOVA 분석
#귀무가설 : 4가지 교육훈련의 성과가 같다.
#대립가설 : 4가지 교육훈련의 성과가 같지 않다.

import scipy.stats as stats
import pandas as pd
import matplotlib.pyplot as plt
import numpy as np
%matplotlib inline
# 32명이 입사를 했는데 각 사원을 그룹으로 나눠서  a,b,c,d 4가지 기법으로 시험을 쳤더니
# 각각의 점수가 나왔다.


# 지금처럼 집단이 3개 이상이면 아노바분석을 사용하면 된다.
### 귀무가설 : 4개의 교육훈련 기법간의 차이가 없다.
### 대립가설 : 4개의 교육훈련 기법간이 차이가 있다.
# 4개의 점수
a = [66,74,82,75,73,97,87,78]
b = [72,51,59,62,74,64,78,63]
c = [61,60,57,60,81,55,70,71]
d = [63,61,76,84,58,65,69,80]

# 4개의 점수에 대한 평균 구하기 
print("a 평균 : ",np.mean(a))
print("b 평균 : ",np.mean(b))
print("c 평균 : ",np.mean(c))
print("d 평균 : ",np.mean(d))

#matplotlib plotting
plot_data=[a,b,c,d]
#박스플롯  : 맨아래가 최소값 :  ㅗ , 맨위가 최대값 : ㅜ 
# 박스안에 대부분의 데이터가 몰려 있다.
# 박스안에 중간에 선은 평균값이다.
# 세번째 즉 c 그룹 같은 경우에는 평균은 낮은데 고득점자가 있다는 걸 
# 볼수 있다. 이렇게 데이터의 분포를 시각화 해서 볼 수 있는 거다.
# 그리고 사분위수를 했었는데 
# 최소값이 0% 지점이고 
# 사분위수 : 사각형 기준으로 아랫면이 25% 지점 , 윗쪽면이 75% 지점 

plt.boxplot(plot_data)
plt.xticks([1,2,3,4],['a', 'b','c','d'])

plt.show()

#f_oneway : 일원배치 분산분석(아노바분석)
F_statistic, pVal = stats.f_oneway(a,b,c,d)
print('F={0:.1f},p={1:.3f}'.format(F_statistic, pVal))

if pVal < 0.05:
    print("대립가설 채택")
else:
    print("귀무가설 채택")
